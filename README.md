# NPTEL-Introduction-to-Programming-in-C-Assignment-1-Question-3
NPTEL Introduction to Programming in C Assignment 1 Question 3

# Question 3
Input : Triplet of three numbers (a,b,c)

Output : 

1 if a>b>c

0, otherwise

# Sample input 1
```sh
1 4 6
```

# Sample output 1
```sh
0
```

# Sample input 2
```sh
1 1 0
```

# Sample output 2
```sh
0
```

# Sample input 3
```sh
1 -1 -4
```

# Sample output 3
```sh
1
```
